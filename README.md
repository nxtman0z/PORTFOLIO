## 🚀 Quick Start  
1. **Clone the repository**
   ```bash
   git clone https://github.com/nxtman0z/portfolio.git
   ```
2. **Navigate to project directory**
   ```bash
   cd portfolio
   ```
3. **Open in browser**
   ```bash
   open index.html
   ```
